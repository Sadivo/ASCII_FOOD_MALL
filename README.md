
一個用純文字 ASCII 藝術打造的互動式美食廣場小遊戲
https://sadivo.github.io/ASCII_FOOD_MALL/

看到網路上別人分享的壽司店小遊戲，於事無聊也做了一個0.0

| 店鋪 | 名稱 | 玩法 |
|------|------|------|
| 🍣 | **煞氣鮨 — 迴轉壽司** | 拖曳食材（鮪魚、鮭魚等）搭配醋飯、海苔，製作壽司 / 刺身 / 手捲 / 丼，支援炸鍋與火槍炙燒 |
| 🍜 | **煉獄軒 — 拉麵店** | 選擇湯底、煮麵（注意計時！）、加配料與辣度，完成後出餐給客人 |
| 🍶 | **黄泉亭 — 居酒屋** | 調酒配串燒，體驗居酒屋文化 |
| 🍰 | **虛無堂 — 甜點店** | 堆疊夢幻甜品 |


```
ASCII_FOOD_MALL/
├── index.html      # 🏬 美食廣場大廳（主頁面）

├── ramen.html      # 🍜 拉麵店
├── izakaya.html    # 🍶 居酒屋
├── dessert.html    # 🍰 甜點店
└── README.md       # 📄 本文件
```
